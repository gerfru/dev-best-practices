# Framework Quick Reference

Quick reference for the four canonical HCI frameworks for UX auditing.

## Microsoft HAX — 18 Guidelines by Phase

| Phase | Guidelines |
|-------|-----------|
| Initially | G1 (Communicate capabilities), G2 (Communicate quality) |
| During | G3 (Timing), G4 (Context relevance), G5 (Social norms), G6 (Bias), G7 (Enable), G8 (Dismiss) |
| When Wrong | G9 (Correction), G10 (Restrict when uncertain), G11 (Explanation) |
| Over Time | G12 (History), G13 (Learning), G14 (Update carefully), G15 (Granular feedback), G16 (Consequences), G17 (Global controls), G18 (Communicate changes) |
